What Is Flowperty ?

Flowperty is a fully responsive and modern Tailwind CSS Landing Page Template, designed specifically for real estate listings, property showcases, and housing businesses. It provides a clean, elegant, and high-performance design that ensures a seamless browsing experience across all devices. Built with the latest Tailwind Css v3.4.17, Flowperty is optimized for fast loading times and easy customization. The template includes pre-defined styles for typography, property cards, and smooth animations, allowing users to effortlessly tailor it to match their branding and listing needs.

Key Features:

  • - Property Focused Design
  • - Dark Mode Support
  • - Fully Responsive & Mobile-Friendly
  • - Smooth Animations with AOS
  • - Google Fonts Support
  • - High Performance
  • - Clean Code
  • - Easy To Customize

Instalation Steps

1. Install Node.js if you don’t have it yet. 2. Run npx tailwindcss init before npm install command if you haven't already installed tailwindcss previously. This will install the npm packages listed in the package.json file. You should now have the project files set up and all the npm packages installed. 3. If you already installed tailwindcss previously, Run npm install then npm run dev dev to start the development server with hot-reloading. This command allows you to preview changes in real-time while customizing your Flowperty template before preparing it for production.

Customization

How to customize colors in Flowperty?
Instead of setting up custom color variables in the tailwind.config.js file, Flowperty applies Tailwind’s utility-first classes directly in the HTML markup—such as bg-gray-900, text-cyan-500, and border-cyan-400. This approach simplifies development by eliminating the need to manage a custom theme and ensures full compatibility with Tailwind’s default palette. If you wish to change a color, you can simply update the class name in the HTML. For more information on available color utilities, check out the Tailwind CSS documentation: Background Color, Text Color, and Border Color.

How to change the font family in Flowperty?
Flowperty makes it easy to customize fonts, First, open the _style.css file (located in `assets/css/style.css`) and import your Google Font: @import url('https://fonts.google.com/')
Next, create a custom class inside `style.css` to define the font family:
.poppins-regular { font-family: "Poppins", sans-serif; font-weight: 400; font-style: normal; }
Finally, apply the font using the class in your HTML:
<p class="poppins-regular">This text uses Poppins font.</p>

How to set up Formspree in Flowperty?
Flowperty uses Formspree to handle contact form submissions easily without a backend. To set it up, follow these steps:
1. Go to Formspree and sign up (if you haven't already).
2. Create a new form and copy the unique Formspree endpoint URL.
3. Open the contactus.html file in Flowperty.
4. Replace the current form's action attribute with your Formspree endpoint.
5. Save the file, and your form is ready to receive submissions via email.
